myHomeCNC v1

    You need to install and run the myHomeCNC Server on the machine that has the GPIO.

    Typically this is installed on the RaspberryPi or BBB or similar device that has GPIO ports connected to devices you will control and part of myHomeCNC project.

    If you already have the server downloaded, you may just need to launch it from the command line.

    Please, ensure the SPJS is also installed and running, with a serial connection established with the main controller.
